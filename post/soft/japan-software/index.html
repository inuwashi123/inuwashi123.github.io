前言

本文介绍下在工作中常用到的软件，主要是日本独有的软件。像万国通用的office全家套之类的就不介绍了。

文本编辑–樱花编辑器（サクラエディタ）

樱花编辑器是日本人开发的强大的文本编辑器，用来代替鸡肋的微软记事本。其地位相当于国内常使用的Notepad++。不仅可以编辑文本，用来徒手写代码也不错。还有宏功能。

樱花编辑器目前只有日文和英文两种版本，没有汉化版。

樱花编辑器官方网站 https://sakura-editor.github.io

Linux,Unix连接工具–Tera Term（テラターム）

Tera Term也是由日本人开发的支持SSH，telnet，串联等协议的连接工具。功能类似于国内常用putty.exe。

Tera Term本身提供了宏功能，能通过编写脚本实现连接，方便简单实用。

Tera Term国际化不错，提供了简体中文，繁体中文版本。有兴趣的去下载试用下。以后有时间的话，会讲解下Tera Term的宏功能，编写宏代码。

Tera Term官方网站 https://ttssh2.osdn.jp

sftp客户端–WinSCP

WinSCP是一款开源的SFTP客户端，运行于Windows系统下，遵照GPL发佈。WinSCP除了SFTP，还支持SSH、SCP。

WinSCP并不是日本开发的，由布拉格经济大学所发展与维护，基于PuTTY的对于SSH的实现，支援 SSH-1 与 SSH-2，WinSCP同时也支援SCP（安全复制）通讯协定，WinSCP具有FTP的档案传输功能，又比FTP更加具有安全性。

由于支持ssh，一般像从linux，unix下载文件是使用WinSCP。WinSCP有汉化版本，国内好多使用。

WinSCP官方网站 https://winscp.net/eng/index.php

文件比较／合并工具–WinMerge

WinMerge是一款运行于Windows系统下的免费的文件比较／合并工具。它非常适用于确定不同版本的文件间的改变以及合并这些改变，尤其是在有多人编辑同一文件的情况下。

WinMerge提供汉化版本。

WinMerge官方网站 https://winmerge.org
